# 🏦 Loan Approval Prediction using Machine Learning

This project explores how machine learning can be used to predict whether a loan application is likely to be approved.

Using financial information such as income, credit score, and loan amount, the model learns patterns from historical data and predicts whether a new loan application might be approved.

The project was built as a machine learning practice project using Python and Google Colab.

---

## 📌 Project Overview

In many financial institutions, loan approval decisions depend on several factors such as income level, credit history, and loan amount.

In this project, we analyze a dataset containing loan-related information and train machine learning models to understand the patterns behind loan approvals.

The goal is to build a model that can help estimate whether a loan application is likely to be approved.

---

## ⚙️ Workflow

The notebook follows a typical machine learning workflow:

• Loading and exploring the dataset
• Cleaning the data and handling missing values
• Removing outliers using the IQR method
• Training different machine learning models
• Comparing model performance
• Visualizing feature importance
• Building a simple prediction interface

---

## 🤖 Machine Learning Models Used

The project experiments with multiple models to compare performance:

* Decision Tree
* Random Forest
* Gradient Boosting

Testing multiple algorithms helps identify which model performs best for this problem.

---

## 💻 Interactive Prediction Interface

A simple interactive interface was created using **Gradio**.
This allows users to enter financial details such as income, credit score, and loan amount and receive a loan approval prediction.

Note:
The Gradio interface runs when the notebook is executed in **Google Colab or a local Python environment**.
GitHub only displays the notebook and code but does not run the interface directly.
